# SkillCrafy
This plugin gives mastery for mining blocks such as trees, stone, and ores, the amount of combing mastery can be changed in config, during the mining at the top will be bossbar. This plugin works on:
1. Core paper;
2. Java 17;
3. Minecraft version 1.16.5.
 
If you have ViaVersion and ViaBackwards installed, then you can log in on versions 1.16.5+ (depending on how you set it up, the plugin will still work.
